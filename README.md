# Blog Sederhana dengan JSON

## Yang diperlukan

json-server

Cara install:

```bash
sudo npm i -g json-server
```

Dia perlu Node minimal versi 10

## Ubah markup

Pakai pug

Install:

```bash
sudo npm i -g pug-cli
```

Terus, jalankan:

```bash
cd pug 
pug . -w -o ../html
```

## Menjalankan

```bash
./jalankan
```

Terus, buka di browser file `html/index.html`
